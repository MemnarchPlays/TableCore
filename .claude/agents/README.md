# Project Agents

Place project-specific Claude Code agent definitions here as `<name>.md` files.

Agents defined here supplement the global agent pool (claude-rails plugin). They are available in this project only.

When to add an agent here:
- The task requires domain knowledge specific to this project (e.g., a Prisma migration reviewer, a Zustand store auditor).
- The task is too specialized for a general-purpose agent but too narrow for a global skill.

See CLAUDE.md for reading hierarchy and project conventions.
