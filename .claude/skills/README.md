# Project Skills

Place project-specific Claude Code skill definitions here as `<name>.md` files.

Skills defined here supplement the global skill pool (claude-rails plugin). They are available in this project only.

When to add a skill here:
- A repeated workflow is project-specific and too narrow for a global skill (e.g., "seed SRD monsters", "run combat simulation test").
- A skill from the global pool needs a project-specific override.
